# 📊 Capstone Project: Road Accident Analysis

## 🎯 Overview
Developed a comprehensive data analysis project focused on identifying patterns, risk factors, and severity drivers in road accidents using Python and Tableau.

---

## 🔷 Key Contributions

### Visualization Design, Dashboard Development & Integration

Led the design and development of interactive Tableau dashboards, focusing on transforming processed data into clear, analytical visualizations.

---

### 📊 Dashboards Built

#### 1. Severity Overview
- Severity Distribution
- Severity by Cause
- Weather × Light Heatmap
- Road Surface vs Severity
- Casualties vs Severity  

#### 2. Time & Pattern Analysis
- Accidents by Hour  
- Day of Week Trends  
- Hour × Day Heatmap  
- Time Period vs Severity  
- Area vs Severity  

---

### 🧠 Key Insights

- Accident patterns are highly time-dependent, with clear peaks during rush hours  
- Severity increases under low visibility and adverse environmental conditions  
- High-risk zones and behaviors can be identified through structured analysis  
- Accidents follow predictable patterns influenced by traffic flow and human behavior
